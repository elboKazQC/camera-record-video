🎯 OBJECTIF : Activer Wolfgang GA100 comme webcam sur votre PC

-------------------------------------------------------------------------------
MÉTHODE 1 : NAVIGATION MENU (La plus courante)
-------------------------------------------------------------------------------

1. Allumez Wolfgang
2. Appuyez sur le bouton MENU
3. Utilisez ⬆️ ⬇️ pour naviguer
4. Cherchez : 
   
   Setup → USB Mode → PC Camera
   OU
   Paramètres → USB Mode → Caméra PC
   OU  
   Settings → Connection → Webcam

5. Sélectionnez "PC Camera" (PAS "Mass Storage")
6. Confirmez avec OK
7. Quittez le menu

RÉSULTAT : Écran Wolfgang devrait afficher "PC Camera" ou "UVC"

-------------------------------------------------------------------------------
MÉTHODE 2 : CONNEXION DIRECTE (Automatique)
-------------------------------------------------------------------------------

1. Wolfgang ÉTEINT
2. Branchez câble USB-C au PC
3. Allumez Wolfgang (pendant qu'il est branché)
4. Wolfgang démarre en mode webcam automatiquement

RÉSULTAT : Windows fait "ding" et détecte la caméra

DÉPANNAGE
-------------------------------------------------------------------------------

PROBLÈME : Je ne trouve pas "USB Mode" dans le menu

SOLUTIONS :
□ Cherchez aussi : "Connection", "System", "Advanced"
□ Termes alternatifs : "UVC", "PC Camera", "Webcam", "USB Setting"
□ Essayez Menu → System → USB
□ Vérifiez version firmware (peut manquer sur anciennes versions)
□ Consultez page du manuel PDF avec Ctrl+F "USB"

PROBLÈME : L'option existe mais Wolfgang n'est pas détecté

SOLUTIONS :
□ Débranchez/rebranchez USB
□ Testez autre port USB (USB 3.0 bleu de préférence)
□ Changez de câble USB-C
□ Redémarrez Wolfgang
□ Redémarrez service Windows : sc start FrameServer
□ Vérifiez dans Gestionnaire de périphériques (devmgmt.msc)

PROBLÈME : Wolfgang détecté mais pas d'image

SOLUTIONS :
□ Retirez cache objectif
□ Fermez autres applications utilisant caméra
□ Vérifiez résolution compatible (essayez 720p)
□ Réinitialisez Wolfgang (Menu → System → Reset)
