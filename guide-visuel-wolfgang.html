🎥 Wolfgang GA100 - Mode Webcam

⚠️ Important : Wolfgang GA100 doit être configuré en mode "PC Camera" pour fonctionner comme webcam.

📋Méthode 1 : Via le Menu (RECOMMANDÉ)

1. Allumez votre Wolfgang GA100
2. Appuyez sur le bouton MENU (ou bouton ⚙️ paramètres)
3. Naviguez avec les boutons ⬆️ ⬇️
4. Cherchez dans cet ordre :
📂 Setup / Paramètres / Settings │ ├── 📹 Video Settings ├── 📸 Photo Settings ├── 🔌 USB Mode ← CHERCHEZ ICI │ │ │ ├── 💾 Mass Storage (Stockage) ❌ NE PAS CHOISIR │ └── 📹 PC Camera / UVC Mode ✅ CHOISIR CECI │ ├── 📡 WiFi Settings └── 🔧 System
5. Sélectionnez PC Camera ou UVC Mode
6. Validez avec le bouton OK/SELECT

🔌Méthode 2 : Connexion Automatique

1. Éteignez Wolfgang GA100
2. Branchez le câble USB-C entre Wolfgang et votre PC
3. Allumez Wolfgang (pendant qu'il est connecté)
4. Wolfgang peut démarrer automatiquement en mode PC Camera

🎮Méthode 3 : Raccourci Boutons

Option A : Wolfgang éteint → Maintenez MODE + POWER simultanément
Option B : Wolfgang allumé et connecté → Appuyez 3 fois rapidement sur bouton MODE

🔍Autres Emplacements Possibles

📂 System / Système └── 🔌 USB Setting └── Camera Mode ✅ 📂 Connection / Connexion └── 🔌 USB Mode └── PC Camera ✅ 📂 Advanced / Avancé └── USB Config └── UVC Mode ✅

✅Comment savoir que ça fonctionne ?

  • Sur Wolfgang : Écran affiche "PC Camera" ou "UVC" ou icône 🎥
  • Sur Windows : Son "ding-dong" de connexion USB
  • Gestionnaire de périphériques : Wolfgang apparaît sous "Caméras"

🚨Toujours pas trouvé ?

  1. Vérifiez la version firmware de Wolfgang (Menu → System → Info)
  2. Certaines versions anciennes peuvent ne pas avoir le mode webcam
  3. Consultez le manuel PDF : cherchez "USB" avec Ctrl+F
  4. Le mode peut s'appeler différemment selon la langue/firmware
  5. Essayez la mise à jour firmware sur le site Wolfgang

💡 Conseil : Si vous voyez votre Wolfgang dans le Gestionnaire de périphériques mais pas l'option dans le menu, c'est que le mode par défaut est peut-être déjà bon. Testez simplement la connexion !
